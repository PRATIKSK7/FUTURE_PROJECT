# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - Production Release - 2026-06-25
### Added
- Complete End-to-End Enterprise Automation.
- CI/CD Pipelines (Playwright, Pytest, Vitest, Trivy).
- Dockerization of Frontend and Backend.
- Storage Abstraction interfaces (S3, Cloudinary).
### Changed
- Promoted all Beta features to stable Release Candidate 1.
- Implemented `React.memo` and Dynamic Imports for production rendering speeds.

## [v0.8.0] - Security & Auth - 2026-06-20
### Added
- Clerk JWT Integration.
- Workspace Isolation (Multi-tenant Architecture).
- Backend RBAC Middleware.

## [v0.7.0] - Persistence - 2026-06-15
### Added
- Neon PostgreSQL Integration.
- SQLAlchemy schemas for Users, Workspaces, and Projects.
- Autosave functionality via React Query mutations.

## [v0.5.0] - The Engine - 2026-06-10
### Added
- The AI Document Workspace (Notion-style rich text).
- Explainability Dashboard for transparent prompt inspection.
- Google Gemini API Streaming Integration.

## [v0.3.0] - Prompt Foundation - 2026-06-05
### Added
- The Context Intelligence Engine.
- Deterministic Prompt Builder.

## [v0.1.0] - Foundation - 2026-06-01
### Added
- Next.js and FastAPI scaffolding.
- Zustand Business Wizard UI.
